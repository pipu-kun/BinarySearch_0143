Binary Search

Cara kerja:
Membagi index ~> mencari bagian yang diyakini ada ~> membuat bagian yang tidak digunakan ~> repeat hingga ketemu.

Keunggulan: 
• Akan cepat prosesnya jika data berada   di tengah index.

Kekurangan: 
• Searching akan terlalu lama dan sia     sia jika data tidak ditemukan.
• Data harus diurutkan dahulu.
